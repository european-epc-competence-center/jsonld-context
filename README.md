# EECC Json-Ld Contexts

This repository shall serve json-ld contexts for eecc puposes under european-epc-competence-center.github.io/jsonld-context/context

## Available Contexts

### TDO Jewelry Context

**Path:** `context/tdo/goldring.jsonld`

Custom JSON-LD context for jewelry and precious metals products, extending Schema.org and GS1 vocabularies.

**Documentation:** See [`context/tdo/README.md`](context/tdo/README.md) for detailed property definitions, usage examples, and TDO-specific information.
